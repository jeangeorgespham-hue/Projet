Description:

Le tri à bulles consiste à parcourir la liste et à comparer les éléments adjacents deux à deux. Si un élément est plus grand que le suivant, ils sont échangés. Ce processus se répète jusqu'à ce que la liste soit entièrement triée.

Optimisation:

Le code inclut une optimisation avec la variable echange. Si, lors d'un passage complet, aucun échange n'est effectué, cela signifie que la liste est déjà triée. L'algorithme s'arrête alors prématurément pour gagner en efficacité.
