Vérificateur de Mot de Passe (Python)
Ce script Python permet de vérifier si un mot de passe respecte des critères de sécurité de base. Il est idéal pour apprendre les rudiments de la manipulation des chaînes de caractères et des boucles en programmation.

Critères de validation
Pour qu'un mot de passe soit considéré comme valide, il doit remplir les conditions suivantes :

Longueur : Au moins 8 caractères.

Majuscule : Contenir au moins une lettre majuscule (A-Z).

Chiffre : Contenir au moins un chiffre (0-9).
