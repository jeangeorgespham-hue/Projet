Suppression des doublons dans un tableau trie
Description
Ce programme implémente une solution efficace pour supprimer les éléments en double dans un tableau d'entiers trié. L'algorithme modifie le tableau directement sur place et retourne le nombre d'éléments uniques.

Fonctionnement
L'algorithme utilise la technique des deux pointeurs. Il parcourt le tableau une seule fois, ce qui permet d'obtenir une performance optimale.

Complexité temporelle : O(n)

Complexité spatiale : O(1)
