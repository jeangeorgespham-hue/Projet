Description
Ce projet contient une solution en C++ pour doubler les zéros dans un tableau d'entiers tout en conservant la taille initiale.

Complexité (Grand O)
Complexité Temporelle : O(n)
La boucle parcourt le tableau une seule fois. Les opérations d'ajout dans le vecteur temporaire sont en temps constant.

Complexité Spatiale : O(n)
L'utilisation d'un vecteur temporaire (temp) nécessite un espace supplémentaire proportionnel à la taille du tableau d'origine.
